# CodeCraftHub

A simple personalized learning platform built with **Python** and **Flask**.  
CodeCraftHub helps developers track courses they want to learn, practice REST API basics, and understand how to store data in a JSON file without using a database.

---

## 📖 Project Overview
CodeCraftHub is designed as a beginner-friendly project to teach REST API fundamentals.  
It allows developers to:
- Add new courses
- View all courses
- Update course details
- Delete courses

All data is stored in a simple `courses.json` file.

---

## ✨ Features
- CRUD operations (Create, Read, Update, Delete) for courses
- JSON file storage (no database required)
- Auto-generated course IDs
- Validation for required fields and status values
- Helpful error messages for beginners
- Simple project structure
